Aula Didática: Operadores Aritméticos em C

Os operadores aritméticos são fundamentais em qualquer linguagem de programação, incluindo C. Eles são usados para realizar operações matemáticas básicas, como soma, subtração, multiplicação e divisão.

Operadores Aritméticos Básicos

Operador Descrição Exemplo
+Somaa + b
-Subtraçãoa - b
*Multiplicaçãoa * b
/Divisãoa / b
%Módulo (resto da divisão)a % b

Operações Básicas


#include <stdio.h>

int main() {
    int a = 10, b = 3;
    int soma, subtracao, multiplicacao, modulo;
    float divisao;

    // Realizando operações
    soma = a + b;
    subtracao = a - b;
    multiplicacao = a * b;
    divisao = (float)a / b;  // (float)a faz a variável A parar de ser INT e começar a ser FLOAT
    modulo = a % b;

    // Exibindo resultados
    printf("Soma: %d + %d = %d\n", a, b, soma);
    printf("Subtração: %d - %d = %d\n", a, b, subtracao);
    printf("Multiplicação: %d * %d = %d\n", a, b, multiplicacao);
    printf("Divisão: %d / %d = %.2f\n", a, b, divisao);
    printf("Módulo: %d %% %d = %d\n", a, b, modulo);

    return 0;
}

Explicação do código acima.

int a = 10, b = 3;
Declara duas variáveis inteiras, a e b, com valores 10 e 3, respectivamente.

int soma, subtracao, multiplicacao, modulo;
Declara quatro variáveis inteiras para armazenar os resultados das operações de soma, subtração, multiplicação e módulo.

float divisao;
Declara uma variável do tipo float para armazenar o resultado da divisão. Utilizamos float para obter resultados com casas decimais.

soma = a + b;
Calcula a soma de a e b (10 + 3) e armazena o resultado (13) na variável soma.

subtracao = a - b;
Calcula a subtração de b de a (10 - 3) e armazena o resultado (7) na variável subtracao.

multiplicacao = a * b;
Calcula a multiplicação de a por b (10 * 3) e armazena o resultado (30) na variável multiplicacao.

divisao = (float)a / b;
Converte a para float para garantir uma divisão real. O resultado da divisão (10.0 / 3) é aproximadamente 3.33 e é armazenado na variável divisao.

modulo = a % b;
Calcula o resto da divisão de a por b (10 % 3) e armazena o resultado (1) na variável modulo. O operador de módulo retorna o resto de uma divisão inteira.

Ordem de Precedência

Assim como na matemática, em C, a ordem das operações importa. A precedência dos operadores determina a ordem em que as operações são realizadas.

  • Parênteses: ()
  • Multiplicação, Divisão e Módulo: *, /, % (da esquerda para a direita)
  • Soma e Subtração: +, - (da esquerda para a direita)

#include <stdio.h>

int main() {
    int resultado;

    resultado = 10 + 5 * 2;    // Multiplicação é feita primeiro
    printf("Resultado: %d\n", resultado);  // 10 + (5 * 2) = 20

    resultado = (10 + 5) * 2;  // Parênteses alteram a ordem
    printf("Resultado com parênteses: %d\n", resultado);  // (10 + 5) * 2 = 30

    return 0;
}

Incremento (++) e Decremento (--)

Estes operadores aumentam ou diminuem o valor de uma variável em uma unidade:

  • Incremento (++): Adiciona 1.
  • Decremento (--): Subtrai 1.

#include <stdio.h>

int main() {
    int x = 5;
    printf("Valor inicial: %d\n", x);

    printf("Pré-incremento: %d\n", ++x);  // x torna-se 6 antes da impressão
    printf("Pós-incremento: %d\n", x++);  // Imprime 6, depois x torna-se 7

    return 0;
}
