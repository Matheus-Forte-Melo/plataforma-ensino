Quebra de Linha em C

Em C, usamos o caractere especial \n para inserir uma quebra de linha, permitindo que o texto seja exibido em múltiplas linhas na saída do console.

Exemplo Prático:


#include <stdio.h>

int main() {
    // Exemplo de uso da quebra de linha
    printf("Linha 1\n");
    printf("Linha 2\n");
    printf("Linha 3\n");

    return 0;
}

Saída no console:

        Linha 1
        Linha 2
        Linha 3

Nota: Você pode usar múltiplos \n no mesmo texto para criar espaçamentos adicionais. Exemplo:


        printf("Linha 1\n\nLinha 3\n");

Nesse caso, haverá uma linha em branco entre "Linha 1" e "Linha 3".

Para que serve o comando #include no início de um programa em C?

Para declarar variáveis globais
Para definir a estrutura do programa
Para incluir bibliotecas que contêm funções prontas
Para finalizar a execução do programa

Sobre o ponto e vírgula (;) no final de linhas de código em C, qual das afirmações abaixo é verdadeira?

É obrigatório no final de declarações e comandos, como printf ou return.
É opcional e usado apenas para organização do código.
Serve para iniciar um novo bloco de código.
Nunca deve ser usado com comandos como #include.
