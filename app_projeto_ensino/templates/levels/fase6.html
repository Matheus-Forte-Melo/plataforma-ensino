Qual é o principal objetivo da lógica de programação?

Criar interfaces bonitas.
Substituir a matemática na programação.
Executar tarefas manuais de forma automática.
Resolver problemas de forma sistemática.

Quais são as características de uma linguagem de programação de alto nível?

É mais próxima da linguagem de máquina e difícil de usar.
É mais próxima da linguagem humana e fácil de entender.
Oferece maior controle sobre o hardware.
É específica para sistemas operacionais.
É usada exclusivamente para jogos.

Sobre a linguagem C, quais afirmações são verdadeiras?

É conhecida por ser apenas uma linguagem de baixo nível.
É amplamente utilizada para aprender conceitos fundamentais de programação.
É eficiente e flexível para diversos tipos de desenvolvimento.
Foi criada exclusivamente para sistemas embarcados.
É base para muitas outras linguagens de programação modernas.

Qual é o papel de um compilador?

Traduzir o código-fonte em linguagem de máquina.
Executar diretamente o código sem tradução.
Facilitar o design de interfaces gráficas.
Permitir a comunicação entre diferentes computadores.
Criar ambientes gráficos interativos.
