O que são condicionais?

São laços de repetição usados para executar um código várias vezes.
São instruções para realizar operações matemáticas no programa.
São funções reservadas para manipular strings no C.
São estruturas que permitem a execução de diferentes blocos de código com base em condições avaliadas como verdadeiras ou falsas.
São funções reservadas para manipular strings no C.

Selecione as afirmações FALSAS sobre as condicionais:

Condicionais raramente aparecem na programação.
A instrução if é obrigatória em todos os programas em C.
Podemos usar operadores lógicos (&&, ||, !) dentro das condicionais.
A instrução else é obrigatória quando usamos um if.
Podemos usar operadores aritiméticos (/, -, ++, [...]) dentro das condicionais.
As condicionais são úteis para implementar estruturas de decisão em programas.
Podemos usar operadores relacionais (>, <, >=, [...]) dentro das condicionais.

#include <stdio.h>

int main() {
    int idade = 20;
    int carteiraValida = 0; 
    int temCarteira = 1; 

    if (idade >= 18 && temCarteira == 1 && carteiraValida == 1) {  
        printf("Pode dirigir\n");
    } else if (idade < 18) {  
        printf("Não pode dirigir: menor de idade\n");
    } else if (!temCarteira) {  
        printf("Não pode dirigir: não possui carteira\n");
    } else if (!carteiraValida) {  
        printf("Não pode dirigir: carteira vencida\n");
    }

    return 0;
}

Encontre o erro lógico desse programa que tem o objetivo de dizer se uma pessoa é adulta ou idosa baseada em sua idade.


                #include <stdio.h>
                
                int main() {
                    int idade = 20;
                
                    if (idade >= 18 && idade <= 65) { // Condicao 1
                        printf("Adulto\n");
                    } else if (idade < 18 || idade > 65) { // Condicao 2
                        printf("Idoso\n");
                    }
                
                    return 0;
                }

O programa não está funcionando corretamente. O que pode ser ajustado?

A segunda condição está incorreta.
A primeira condição deve ser idade > 18 && idade < 65.
O else if é desnecessário, pois cobre todas as possibilidades da primeira condição.
O programa na verdade está correto.
