Fases

  • Conteúdo: Esta é uma fase dedicada à teoria. Aqui, você aprenderá conceitos importantes que servirão de base para os próximos desafios, tal qual sugestões de Exercicios Opcionais para você práticar. Essas fases são representadas por um círculo verde.

  • Exercício: Teste seu aprendizado com questões objetivas! Para avançar, você precisa acertar todas as questões, mas não se preocupe! você tem tentativas infinitas (Cuidado para não falhar TANTAS vezes, você vai ganhar menos pontos). Essas fases são representadas por um quadrado vermelho.

  • Desafio: Fases opcionais, mas altamente recomendadas! Elas são mais desafiadoras e exigem pensamento criativo, muitos pontos. Essas fases são representadas por um losango amarelo.

  • Prova: São como chefões de um jogo. Você precisará responder corretamente a todas as questões para avançar, sem consultas, com um temporizador e valendo NOTA. Você poderá tentar várias vezes, mas só verá o resultado final quando conseguir acertar o determinado número de questões para concluir a prova. Essas fases são representadas por um hexágono laranja.

Como navegar pelo mapa

  • Para fechar ou minimizar: Use os botões no topo da página ( para fechar e para minimizar).
  • No PC: Clique e arraste para navegar pelo mapa. Para recentralizar, pressione a barra de espaço.
  • No celular: Navegue arrastando a tela com um dedo. Para recentralizar o mapa, dê um duplo toque na tela.

O que você vai conseguir fazer de olhos fechados até o fim do curso:


    #include <stdio.h>

    int main() {
        char operador;
        double num1, num2, resultado;
    
        printf("Calculadora simples em C\n");
        printf("Escolha uma operação (+, -, *, /): ");
        scanf(" %c", &operador);
    
        printf("Digite o primeiro número: ");
        scanf("%lf", &num1);
    
        printf("Digite o segundo número: ");
        scanf("%lf", &num2);
    
        switch (operador) {
            case '+':
                resultado = num1 + num2;
                printf("Resultado: %.2lf\n", resultado);
                break;
            case '-':
                resultado = num1 - num2;
                printf("Resultado: %.2lf\n", resultado);
                break;
            case '*':
                resultado = num1 * num2;
                printf("Resultado: %.2lf\n", resultado);
                break;
            case '/':
                if (num2 != 0) {
                    resultado = num1 / num2;
                    printf("Resultado: %.2lf\n", resultado);
                } else {
                    printf("Erro: Divisão por zero não é permitida.\n");
                }
                break;
            default:
                printf("Erro: Operação inválida.\n");
        }
    
        return 0;
    }
