Estruturas Condicionais: if, else if e else

As estruturas condicionais são fundamentais para tomar decisões na programação. Elas utilizam as condições formadas por operadores relacionais e lógicos (podendo até mesmo serem combinados com operadores aritiméticos, variáveis e constantes) para determinar qual bloco de código será executado.

Estruturas Básicas

A estrutura condicional permite verificar uma ou mais condições e executar diferentes blocos de código com base nos resultados:

  • if: Usado para avaliar uma condição específica. Se a condição for true, o código associado ao if será executado.
  • else if: Usado para verificar condições adicionais caso a primeira (ou outras) não seja verdadeira.
  • else: Define um bloco de código que será executado se nenhuma das condições anteriores for verdadeira.

Exemplo Simples

Verificar se um número é positivo, negativo ou zero:


    int numero = -5;
    
    if (numero > 0) {
        printf("O número é positivo.");
    } else if (numero < 0) {
        printf("O número é negativo.");
    } else {
        printf("O número é zero.");
    }

Como funciona

O if avalia a primeira condição:

  • numero > 0: Usamos o operador relacional > para verificar se o número é maior que zero. Se for verdadeiro, a mensagem "O número é positivo." será exibida.

Se a primeira condição for falsa, o else if será avaliado:

  • numero < 0: Aqui usamos o operador relacional < para verificar se o número é menor que zero. Se for verdadeiro, a mensagem "O número é negativo." será exibida.

Se nenhuma das condições anteriores for verdadeira, o else será executado, exibindo a mensagem "O número é zero."

Referência aos Operadores

Este exemplo utiliza os operadores relacionais para criar as condições:

  • >: Maior que, para verificar números positivos.
  • <: Menor que, para verificar números negativos.

Note que essas condições resultam em valores booleanos (true ou false), os mesmos que podem ser combinados com operadores lógicos, se necessário.

Exemplo Ampliado

Imagine que você também quer verificar se o número está entre 1 e 10:


    int numero = 5;
    
    if (numero > 0 && numero <= 10) {
        printf("O número está entre 1 e 10.");
    } else if (numero < 0) {
        printf("O número é negativo.");
    } else {
        printf("O número é maior que 10 ou igual a zero.");
    }

Aqui usamos o operador lógico && (AND) para combinar duas condições: numero > 0 e numero <= 10. Ambas devem ser true para que o bloco seja executado.

Resumo

As estruturas condicionais em conjunto com operadores relacionais e lógicos permitem:

  • Tomar decisões com base em condições simples ou complexas.
  • Organizar o fluxo do programa de forma clara e lógica.
  • Executar diferentes blocos de código dependendo do estado das variáveis.

Essa é a base para qualquer aplicação que precise de lógica condicional, desde sistemas básicos até algoritmos complexos.

  1. Verificação de Idade:

    Escreva um programa que solicite a idade de uma pessoa e exiba se ela é criança (0-12 anos), adolescente (13-17 anos), adulta (18-59 anos) ou idosa (60 anos ou mais).

  2. Classificação de Notas:

    Solicite a nota de um aluno (0 a 10). Mostre a classificação conforme:

    • Nota >= 9: Excelente
    • Nota >= 7: Bom
    • Nota >= 5: Regular
    • Nota < 5: Insuficiente

  3. Verificação de Número Par ou Ímpar:

    Peça ao usuário para inserir um número e determine se ele é par ou ímpar.

  4. Calculadora Simples:

    Desenvolva um programa que solicite dois números e uma operação (+, -, *, /). Use condicionais para realizar a operação indicada e exiba o resultado.

  5. Verificador de Maior Número:

    Peça três números ao usuário e exiba o maior entre eles.

  6. Determinação de Triângulo:

    Solicite ao usuário três lados de um triângulo. Verifique se é equilátero (todos os lados iguais), isósceles (dois lados iguais) ou escaleno (todos diferentes).

  7. Classificação de Temperatura:

    Peça ao usuário para inserir a temperatura em graus Celsius. Exiba:

    • Menor que 0°C: Muito frio
    • 0°C a 15°C: Frio
    • 16°C a 25°C: Agradável
    • 26°C a 35°C: Quente
    • Acima de 35°C: Muito quente

  8. Calculadora de IMC:

    Solicite o peso (kg) e a altura (m) do usuário e calcule o IMC. Classifique o resultado:

    • IMC < 18.5: Abaixo do peso
    • 18.5 <= IMC < 25: Peso normal
    • 25 <= IMC < 30: Sobrepeso
    • IMC >= 30: Obesidade
