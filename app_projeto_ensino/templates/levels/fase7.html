Estrutura Básica de um Programa em C

Estrutura Fundamental

Todo programa em C segue uma estrutura básica que deve ser compreendida para criar e organizar códigos corretamente. Vamos explorar os elementos principais:

{% include 'partial/aviso.html' with tipo="vermelho" mensagem="IMPORTANTE: Toda linha de código deve terminar com ';' ponto e vírgula! Isso é MUITO importante, pois indica para o compilador onde uma linha de código acaba." %}
  • Função main(): É o ponto de entrada de qualquer programa em C. Quando você executa seu programa, o computador começa a executar as instruções a partir da função main().
  • #include e Bibliotecas: A palavra-chave #include é usada para adicionar bibliotecas que fornecem funções pré-definidas. Por exemplo, #include <stdio.h> inclui funções como printf() e scanf().

Exemplo Básico:


// Este é um programa simples em C
#include <stdio.h>  // Inclui a biblioteca padrão de entrada e saída

int main() {
    printf("Olá, Mundo!");  // Exibe uma mensagem na tela
    return 0;  // Indica que o programa terminou com sucesso
}

Explicação do Código:

  • #include <stdio.h>: Adiciona a biblioteca que contém a função printf().
  • int main(): Declara a função principal do programa.
  • printf("Olá, Mundo!");: Imprime "Olá, Mundo!" na tela.
  • return 0;: Indica que o programa foi executado com sucesso.

Comentários no Código

Comentários são usados para documentar e explicar o código. Eles são ignorados pelo compilador, servindo apenas para o programador. Existem dois tipos de comentários em C:

  • Comentários de Linha: Usam // para comentar uma única linha.
  • Comentários de Bloco: Usam /* */ para comentar múltiplas linhas.

Exemplo:


    #include <stdio.h>

int main() {
    // Este é um comentário de linha
    printf("Este é um exemplo de comentário de linha.\n");

    /*
        * Este é um comentário de bloco.
        * Ele pode ser usado para documentar várias linhas.
        */
    printf("Este é um exemplo de comentário de bloco.\n");

    return 0;
}

Comentários são uma prática recomendada, especialmente em códigos complexos. Eles ajudam você (e outras pessoas) a entender a lógica do programa no futuro.

Primeiro Programa: "Hello, World"

O "Hello, World" é tradicionalmente o primeiro programa escrito em qualquer linguagem. Ele nos ensina a:

  • Configurar o ambiente de desenvolvimento.
  • Compilar e executar um programa básico.

Exemplo Completo:


    // Programa "Hello, World" em C
    #include <stdio.h>
    
    int main() {
        printf("Hello, World!\n");  // Exibe "Hello, World!" na tela
        return 0;
    }

Como Compilar e Executar:

  1. Abra o DEV-C++ ou o terminal (se estiver usando outro editor).
  2. Copie e cole o código acima no editor.
  3. Salve o arquivo com a extensão .c, por exemplo, hello_world.c.
  4. Compile o programa. No DEV-C++, clique em "Compilar e Executar" ou pressione F11.
  5. Observe o resultado na janela de saída: Hello, World!.
