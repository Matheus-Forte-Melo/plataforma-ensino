Entrada de Dados

A entrada de dados em C é realizada por meio da função scanf. Essa função permite capturar dados fornecidos pelo usuário e armazená-los em variáveis, tornando o programa interativo.

Como usar scanf

O scanf é a principal função para conseguirmos enviar dados para o computador de forma dinâmica. Podemos enviar Números, Datas, Nomes (Apesar de ser um pouco mais avançado), e muito mais. Só precisamos especificar o tipo e a variável na qual queremos atribuir o valor digitado.

scanf("%{especificador de tipo}", &{variável})

Exemplo simples:

#include <stdio.h>
    
            int main() {
                int idade;
            
                // Solicitar a entrada do usuário
                printf("Digite sua idade: ");
                scanf("%d", &idade);
            
                // Exibir a saída
                printf("Sua idade é %d anos.\n", idade);
            
                return 0;
            }

Linha por Linha

  • #include <stdio.h>: Inclui a biblioteca padrão de entrada e saída. Essa biblioteca fornece funções como printf e scanf.

  • int main(): Define a função principal, onde o programa começa sua execução. O tipo int indica que a função retornará um valor inteiro.

  • int idade;: Declara uma variável inteira chamada idade para armazenar a idade inserida pelo usuário.

  • printf("Digite sua idade: ");: Exibe no terminal a mensagem "Digite sua idade: ", solicitando que o usuário insira um número.

  • scanf("%d", &idade);: Captura o valor digitado pelo usuário e o armazena na variável idade.
    • "%d": Especificador de formato para números inteiros.
    • &idade: Passa o endereço da variável idade, permitindo que o scanf armazene diretamente o valor nela.

  • printf("Sua idade é %d anos.\\n", idade);: Exibe a idade armazenada na variável idade. O %d é substituído pelo valor da variável.

  • return 0;: Retorna 0, indicando que o programa foi executado com sucesso.

Fluxo do Programa

  1. O programa exibe a mensagem "Digite sua idade: " no terminal.
  2. O usuário digita um número (ex.: 25) e pressiona Enter.
  3. A função scanf lê o valor digitado e o armazena na variável idade.
  4. O programa exibe a mensagem "Sua idade é 25 anos.", substituindo %d pelo valor de idade.
  5. O programa retorna 0 e é finalizado com sucesso.

Especificadores de Formato

A tabela abaixo mostra os principais especificadores usados com o scanf:

Especificador Tipo de Dado Exemplo de Declaração
%d Inteiros (base decimal) int idade;
%i Inteiros (base automática) int numero;
%f Números de ponto simples (float) float altura;
%lf Números de ponto complexo (double) double nanometro;

Constantes em C

Constantes são valores que permanecem fixos durante toda a execução do programa. Diferente das variáveis comuns, que podem ser redeclaradas e reatribuidas.

As constantes são declaradas com a palavra-chave const ou podem ser definidas com #define.

Declaramos uma constante da seguinte forma

Adendo: Apesar de não ser necessário, geralmente constantes devem ser declaradas em letra maiúscula. Para que possamos à distinguir de uma variável comum.

-#define:

#define LIMITE_MAXIMO 100

Exemplo Completo

O programa a seguir demonstra o uso de entrada de dados, constantes, declaração de variáveis e saída de dados, ou seja, tudo que vimos até agora. Pode parecer desafiador, mas eu te convido a estudar esse código e tentar replicá-lo.


    
// Inclusão da biblioteca padrão       
#include <stdio.h>

// Definição de constante
#define float ALTURA_MINIMA = 1.50;

int main() {
    char inicial;
    float altura;

    // Entrada de dados
    printf("Digite a inicial do seu nome: ");
    scanf(" %c", &inicial); // O espaço antes de %c evita problemas ao capturar o caractere.

    printf("Digite a sua altura em metros (ex: 1.75): ");
    scanf("%f", &altura);

    // Saída de dados
    printf("\nOlá, %c! Sua altura é %.2f metros.\n", inicial, altura);
    printf("A altura mínima definida no programa é %.2f metros.\n", ALTURA_MINIMA);

    return 0;
            
}
