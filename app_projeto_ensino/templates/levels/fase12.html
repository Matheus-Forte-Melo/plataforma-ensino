Identifique o erro no código abaixo:

#include <stdio.h>

int main() {
    float num;
    printf("Digite um número: ");
    scanf("%d", &num);  // ERRO: Deve ser %f para float
    printf("Número digitado: %f\n", num);
    return 0;
}
O código compila e executa corretamente.
O formato especificador em scanf está incorreto para o tipo float.
A variável num não foi declarada corretamente.
