Tipos de Dados e Saída de Dados

Tipos Primitivos

Em C, os tipos de dados primitivos representam as unidades básicas para o armazenamento de informações. Eles definem tanto o tipo de dado que uma variável pode conter quanto o espaço de memória necessário para armazená-lo. A seguir, detalhamos os principais tipos primitivos disponíveis em C:

Tipo Tamanho (aproximado) Descrição Exemplo de Valor
int 4 bytes Utilizado para armazenar números inteiros, como idades, contagens ou qualquer valor numérico sem parte decimal. 10, -50, 0
float 4 bytes Usado para armazenar números com casas decimais, sendo ideal para variáveis como altura, peso e outras medições que exigem precisão limitada. 3.14, -7.5
double 8 bytes Semelhante ao float, mas com maior precisão, utilizado para cálculos mais complexos e que demandam maior exatidão, como cálculos científicos ou financeiros. 3.141592, -0.0001
char 1 byte Armazena um único caractere, como letras, símbolos ou números que serão tratados como texto. 'A', 'z', '1'

Cada tipo de dado foi projetado para atender a necessidades específicas e otimizar o uso da memória. Por isso, é fundamental escolher o tipo adequado conforme o tipo de dado que você está manipulando, a fim de evitar desperdícios de espaço e garantir a precisão das operações.

Declaração de Variáveis

Para armazenar dados, é necessário declarar variáveis. Uma variável é uma área reservada na memória do computador, associada a um tipo de dado, onde um valor pode ser armazenado e posteriormente manipulado. Vamos ver como declarar variáveis e atribuir valores iniciais: 


#include <stdio.h>

int main() {
    int idade = 25;         // Declara uma variável inteira e a inicializa com o valor 25
    float altura = 1.75;    // Declara uma variável float e a inicializa com 1.75
    char inicial = 'M';     // Declara uma variável char e a inicializa com 'M'
    double tamanho_cabelo;  // Declara uma variável double e não a inicializa. (Está definida, mas sem um valor)

    double tamanho_cabelo = 5.5213 // Atribui um valor à essa variável.

    return 0;
}

Neste exemplo, temos três variáveis sendo declaradas:

  • int idade = 25;: Aqui, a variável idade é declarada como do tipo int e recebe o valor 25.
  • float altura = 1.75;: A variável altura é do tipo float e é inicializada com o valor 1.75, que é um número decimal.
  • char inicial = 'M';: A variável inicial armazena um caractere único, neste caso, a letra 'M'.
  • double tamanho_cabelo; Essa variável é declarada, mas só posteriormente atribuímos um valor à ela.

Na programação, é extremamente comum a declaração de variáveis nulas, sem nenhua atribuição de dados. Então, fique ligado.

Saída básica de dados

Okay, os dados estão na memória, mas como fazemos para acessá-los? É simples. Mas antes, saiba que chamamos o processo de retorno de dados da memória de saída de dados

Função printf

A função printf é essencial para a saída de dados em C (ou seja, o acesso de dados). Ela permite imprimir textos e valores armazenados nas variáveis na tela. A grande vantagem dessa função é a possibilidade de formatar a saída, ajustando a forma como os dados são exibidos.

Observe abaixo o retorno dos dados guardados em memória anteriormente.

        
#include <stdio.h>

int main() {
    int idade = 25;        
    float altura = 1.75;   
    char inicial = 'M';   
    
    printf("A sua idade é: %i", idade); // Saída no console: A sua idade é 25. 
    printf("Você tem %f metros de altura", altura); // Saída no console: Você tem 1.75000 metros de altura
    printf("%c é sua incial", inicial); // Saída no console: M é sua inicial

    return 0;
}

O printf permite a formatação dos valores de forma simples. No exemplo acima:

  • printf("A sua idade é: %i", idade); %i ou %d representa o TIPO INT.
  • printf("Você tem %f metros de altura", altura); %f representa o TIPO FLOAT. Se você quiser cortar as casas decimais, troque f por .1f, .2f, .3f e assim por diante
  • printf("%c é sua incial", inicial); %c representa o TIPO CHAR
Note que no lugar onde estaria a varíavel, no printf, temos uma porcentagem seguida por um caractere, em todas as ocasiões. Estas representações se chamam especificadores da dados, e são importantíssimos para avisar ao computador ONDE o dado deve estar, e qual o seu tipo na saída.
