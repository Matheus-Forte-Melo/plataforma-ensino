As bibliotecas que utilizamos geralmente são a stdio.h e stdlib.h.

  • stdio.h: Focada em operações de entrada e saída, incluindo leitura/escrita de dados no console e manipulação de arquivos.
  • stdlib.h: Fornece funções para manipulação de memória, conversão de tipos de dados, controle de processo, ordenação, busca, e geração de números aleatórios.

Essas bibliotecas são fundamentais e amplamente utilizadas na programação em C.

                int main() {

Objetivo: Definir a função principal do programa onde a execução começa. Todo programa deve ter uma função main.

3. Declaração de variáveis:

                float fahrenheit, celsius;

Objetivo: Declarar duas variáveis do tipo float para armazenar a temperatura em fahrenheit e a temperatura convertida em celsius.

4. Solicitação da temperatura em fahrenheit:

                printf("DIGITE A TEMPERATURA EM GRAUS FAHRENHEIT (°F): ");

Objetivo: Exibir uma mensagem para o usuário, pedindo que insira a temperatura em fahrenheit.

                scanf("%f", &fahrenheit);

Objetivo: Ler o valor inserido pelo usuário e armazená-lo na variável fahrenheit, %f é o especificador de formato para números de ponto flutuante.

                fflush(stdin);

Objetivo: Limpar o buffer de saída.

5. Conversão da temperatura:

Sabendo que a fórmula para converter celsius para fahrenheit é:

                Celsius = 5 * (fahrenheit - 32) / 9;

Objetivo: Aplicar a fórmula de conversão de fahrenheit para celsius.

Explicação:

  • Subtrair 32: Ajusta a base da escala fahrenheit para a base celsius.
  • Multiplicar por 5: Escala a diferença para o sistema celsius.
  • Dividir por 9: Ajuda a proporção correta entre duas escalas de temperatura.

6. Exibição do resultado:

                printf("A temperatura em graus Celsius (°C) é: %.2f\n", celsius);

Objetivo: Exibir a temperatura convertida em celsius na tela, %.2f especifica que o número deve ser formatado com duas casas decimais.

7. Fim da função principal:

                return 0;
                }

Objetivo: Indicar que o programa terminou sua execução com sucesso.

  • Preparar o ambiente: Inclua bibliotecas necessárias;
  • Inicializar o programa: Defina a função main;
  • Declarar variáveis: Armazene as temperaturas;
  • Obter entrada: Peça ao usuário para inserir a temperatura em fahrenheit;
  • Converter: Use a fórmula para converter fahrenheit para celsius;

Declaração de variáveis

Declaração de Variáveis

Imagine uma caixa que você pode armazenar coisas dentro dela, você tem a caixa e quer guardar algo dentro dela, como um brinquedo. Essa caixa pode ter um rótulo, assim você sabe o que tem lá dentro. Na programação, uma variável é como essa caixa: você pode guardar informações dentro dela e dar um nome para saber o que está lá.

Como funciona?

  • Nome da variável: É como o rótulo da caixa. Você escolhe um nome que faz sentido. Por exemplo, se você vai guardar a idade de uma pessoa, pode chamar a variável de idade.
  • Valor da variável: É o que você coloca dentro da caixa. Continuando com o exemplo, se a pessoa tem 25 anos, você coloca o número 25 na variável idade.
  • Declaração: Para criar a variável, você usa uma instrução que informa ao computador que está criando essa "caixa". Por exemplo, em muitas linguagens de programação, você escreveria algo como: idade = 25 (idade é o nome da variável e 25 é o valor que estamos guardando).

Resumindo:

  • Variável: Caixa para armazenar informações.
  • Nome: Rótulo da caixa.
  • Valor: O que está dentro da caixa.
  • Declaração: Ação de criar a caixa.

Assim, quando você precisa usar essa informação depois, basta referenciar a variável pelo nome. Se você quiser saber a idade, pode simplesmente usar idade e o computador vai lembrar que é 25.

Comandos de Entrada e Saída

Entrada

É quando você quer que o computador receba informações do usuário. Por exemplo, você pode querer saber o nome ou a idade de alguém.

Saída

É quando o computador mostra informações para o usuário. Por exemplo, pode mostrar uma mensagem de saudação ou a idade que foi informada.

Como isso funciona em C?

Saída com printf

printf é um comando usado para mostrar informações na tela. Imagine que você quer cumprimentar alguém. Você pode escrever: 

printf("Olá, mundo!\n");

Aqui, o computador vai mostrar na tela "Olá, mundo!". O \n é uma maneira de dizer ao computador para pular para a linha de baixo depois de mostrar a mensagem.

Entrada com scanf

scanf é um comando usado para receber informações do usuário. Por exemplo, se você quer saber a idade de alguém, pode fazer assim: 

int idade;

            printf("Digite sua idade: ");
            scanf("%d", &idade);

Aqui, o que acontece é:

  • printf mostra a mensagem "Digite sua idade:" na tela.
  • scanf espera que o usuário digite um número. O %d indica que estamos esperando um número inteiro, e o &idade diz ao computador onde guardar esse número (na variável idade).

Resumindo

  • Entrada: Use scanf para receber (ler) informações do usuário.
  • Saída: Use printf para mostrar informações na tela.

Exemplo Completo

Aqui está um pequeno exemplo que combina tudo isso: 

#include <stdio.h>

            int main() {
                int idade;
                
                printf("Digite sua idade: ");      // Saída
                scanf("%d", &idade);               // Entrada
                
                printf("Você tem %d anos.\n", idade); // Saída
                return 0;
            }

Nesse exemplo:

  • O programa pede a idade do usuário.
  • O usuário digita a idade.
  • O programa mostra quantos anos a pessoa tem.

Estruturas de Controle

Condicionais

Permitem que o programa tome decisões com base em condições. Exemplo: if, else, switch.

If (Se)

A estrutura if é usada para executar um bloco de código apenas se uma condição especificada for verdadeira.


            if (condição) {
                // código a ser executado se a condição for verdadeira
            }

Exemplo em C:


            int idade = 18;

            if (idade >= 18) {
                printf("Você é maior de idade.\n");
            }

Else (Senão)

A estrutura else é usada em conjunto com if para executar um bloco de código quando a condição do if não é verdadeira.


            if (condição) {
                // código se a condição for verdadeira
            } else {
                // código se a condição for falsa
            }

Exemplo em C:


            int idade = 16;

            if (idade >= 18) {
                printf("Você é maior de idade.\n");
            } else {
                printf("Você é menor de idade.\n");
            }

Else if (Senão se)

Usado quando você tem várias condições diferentes de respostas. Ele permite testar várias condições de respostas em sequência.


            if (condição1) {
                // código se a condição1 for verdadeira
            } else if (condição2) {
                // código se a condição2 for verdadeira
            } else {
                // código se nenhuma das condições anteriores for verdadeira
            }

Exemplo em C:


            int nota = 85;

            if (nota >= 90) {
                printf("Nota excelente.\n");
            } else if (nota >= 70) {
                printf("Nota boa.\n");
            } else {
                printf("Nota insuficiente.\n");
            }

Switch

A estrutura switch é usada para selecionar um bloco de código a ser executado com base no valor de uma expressão. É uma alternativa ao uso de múltiplos if e else quando há várias condições baseadas em um único valor.


            switch (variável) {
                case valor1:
                    // Código a ser executado se variável == valor1
                    break;
                case valor2:
                    // Código a ser executado se variável == valor2
                    break;
                // Mais casos...
                default:
                    // Código a ser executado se nenhum dos valores corresponder
            }

Componentes do switch:

  • switch (variável): Avalia o valor da variável.
  • case valor: Define um valor que, se corresponder ao valor da variável, o código associado será executado.
  • break: Encerra o bloco do case.
  • default: Executado se nenhum dos valores case corresponder ao valor da variável.

Exemplo Prático:


            #include <stdio.h>

            int main() {
                int dia = 3; // suponha que o valor 3 representa Quarta-feira

                switch (dia) {
                    case 1:
                        printf("Segunda-feira\n");
                        break;
                    case 2:
                        printf("Terça-feira\n");
                        break;
                    case 3:
                        printf("Quarta-feira\n");
                        break;
                    case 4:
                        printf("Quinta-feira\n");
                        break;
                    case 5:
                        printf("Sexta-feira\n");
                        break;
                    case 6:
                        printf("Sábado\n");
                        break;
                    case 7:
                        printf("Domingo\n");
                        break;
                    default:
                        printf("Número do dia inválido\n");
                        break;
                }

                return 0;
            }

O Que Está Acontecendo?

  • switch (dia): O valor de dia (3) será comparado com cada case.
  • case 1: até case 7: Cada caso verifica se o valor da variável corresponde ao número associado.
  • case 3: Quando dia é 3, este bloco de código é executado e imprime "Quarta-feira".
  • break: Após imprimir "Quarta-feira", o break faz com que o programa saia do switch.
  • default: Se dia não corresponder a nenhum valor entre 1 e 7, o default será executado e imprimirá "Número do dia inválido".

Operadores Lógicos

Muitas vezes, você precisará combinar várias condições usando operadores lógicos como: && (E), || (OU) e ! (NÃO).

Exemplo:


                    if (condição1 && condição2) {
                        // código se ambas as condições forem verdadeiras
                    }

Exemplo em C:


                    int idade = 20;
                    int carteira = 1; // 1 significa que tem carteira de motorista

                    if (idade >= 18 && carteira == 1) {
                        printf("Você pode dirigir.\n");
                    } else {
                        printf("Você não pode dirigir.\n");
                    }

Resumindo

  • Condicionais permitem que o programa escolha entre diferentes ações com base em condições.
  • if: Executa código se a condição for verdadeira.
  • else: Executa código se a condição do if for falsa.
  • else if: Permite testar várias condições.
  • Operadores lógicos: Combinam várias condições para testes mais complexos.
